2. Relational Operators

These operators compare two values.

They always return true or false.

| Operator | Meaning          |
| -------- | ---------------- |
| `==`     | Equal            |
| `!=`     | Not Equal        |
| `>`      | Greater Than     |
| `<`      | Less Than        |
| `>=`     | Greater or Equal |
| `<=`     | Less or Equal    |
