# Day 001 Notes - Java Basics

## What is Java?

Java is a high-level, object-oriented programming language. It is used to build backend applications, web applications, Android apps, enterprise systems, and Spring Boot applications.

## Why Java?

Java is popular because:
- It is platform independent
- It is object-oriented
- It is secure
- It is used in enterprise companies
- It works very well with Spring Boot and microservices

## JDK, JRE, JVM

### JVM

JVM means Java Virtual Machine. It runs Java bytecode.

### JRE

JRE means Java Runtime Environment. It provides the environment required to run Java programs.

### JDK

JDK means Java Development Kit. It is used to develop Java applications. It contains JRE, compiler, and development tools.

## Java Execution Flow

Java source code is saved in a `.java` file.

Example:

java
HelloWorld.java
